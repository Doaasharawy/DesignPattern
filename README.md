# DesignPattern Simple Calculation App
## ( MVC - MVP - MVVM ) & Data Binding
## app contain three buttons to make calculations between two Numbers and show result on textView
### 1- Plus Button  
using MVC Pattern 
### 2- Division Button  
using MVP Pattern 
### 3- Multiplication Button  
using MVVM Pattern
